# Sales Performance Breakdown (SQL Project)
# Objective
Analyze sales performance using SQL with Common Table Expressions (CTEs).

# Tools Used
- SQL (CTE, Window Functions)
- SQLite / MySQL

# Key Analysis
- Total sales by region
- Top 3 products per region
- Monthly revenue trends

# Key Insights
- Certain regions dominate overall sales
- Top-performing products vary by region
- Seasonal trends impact revenue

# Skills Demonstrated
- CTE usage
- Window functions (RANK)
- Aggregations and grouping

# Business Insights
- West region generates highest revenue
-Phones and Laptops dominate top sales
-Revenue peaks in festive months
-Some regions depend heavily on 1–2 products
